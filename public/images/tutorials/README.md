# Tutorial Screenshots

This folder contains screenshots for the Help/Lessons section.

## Naming Convention

Screenshots should be named using the lesson ID followed by the step number:
- Format: `{lessonId}-step{stepNumber}.png` or `.jpg`
- Example: `create-animals-step1.png`, `create-animals-step2.png`

## Adding New Screenshots

1. Take a clear screenshot of the relevant UI element
2. Crop/edit to highlight the important parts
3. Save with the appropriate naming convention
4. Place in this folder
5. Screenshots will automatically appear in the Help section
