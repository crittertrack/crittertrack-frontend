- Green (#16A34A) - Active, success, available
- Red (#DC2626) - Suspended, failed, critical

## Responsive Breakpoints

```
Mobile (< 640px):
- Single column layout
- Stacked navigation
- Full-width tables with scroll
- Simplified modals

Tablet (640px - 1024px):
- Two column layout
- Sidebar navigation
- Multi-column tables
- Standard modals

Desktop (> 1024px):
- Full feature display
- All columns visible
- All sections accessible
- Full-featured dialogs
```
